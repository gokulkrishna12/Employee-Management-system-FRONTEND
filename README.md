# Employee Management System - Frontend

A modern Employee Management System frontend built using React.js. This application provides a user-friendly interface for managing employee records through CRUD operations integrated with a Spring Boot backend.

---

## Features

* View All Employees
* Add New Employee
* Update Employee Information
* Delete Employee
* Responsive User Interface
* REST API Integration
* Real-Time Data Updates

---

## Tech Stack

### Frontend

* React.js
* Vite
* Bootstrap
* Axios
* React Router DOM

---

## Functionalities

### Employee List

* Displays all employee records in a table format.

### Add Employee

* Create a new employee by entering:

  * First Name
  * Last Name
  * Email

### Update Employee

* Modify existing employee information.

### Delete Employee

* Remove employee records instantly.

---

## Installation

### Clone Repository

```bash
git clone <repository-url>
```

### Install Dependencies

```bash
npm install
```

### Run Application

```bash
npm run dev
```

Application will run on:

```text
http://localhost:5173
```

---

## Backend Integration

This frontend communicates with a Spring Boot REST API running on:

```text
http://localhost:8080/api/employees
```
